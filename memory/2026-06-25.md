# 2026-06-25

## Связь: международная eSIM для нового телефона (подбор)

### Контекст
Тема связи в Грузии (роуминг МТС «Комфорт») решена ранее (24 июня). Новый запрос Can: новый телефон с поддержкой eSIM пока без симки (физсимка нужна в другом телефоне). Хочет завести **международную eSIM** на новый телефон. Телефон в Грузию НЕ берёт (принцип «минимум вещей, не привлекать воров») — остаётся дома в РФ.

### Требования Can
- **Международная** (не российская), чтобы в перспективе поехать с ней за границу.
- **Долгоживущая** (не туристическая «на месяц») — несгораемый баланс.
- **Главная цель** (уточнилась в ходе беседы): обход блокировок из РФ через зарубежный IP. Работать в РФ обязана.
- Звонки **не критичны** — достаточно интернета.
- **Оплата по приоритету:** 1) крипта, 2) СБП, 3) виртуальная межд. карта (для подписок, есть), 4) карта РФ, 5) полноценная межд. карта (есть).

### Что разобрали
**Ключевое разделение:** «eSIM» = два разных продукта.
1. eSIM росоператора (МТС и т.п.) — росномер, звонки, в РФ родная, за границу = роуминг. Не подходит (нужна международная).
2. Иностранная/международная eSIM — почти всегда только интернет, без номера/звонков. Это и нужно Can.

**Критичный параметр для обхода блокировок — тип IP.** Не все иностранные eSIM дают зарубежный IP; часть работает через росроуминг-партнёра и выдаёт российский IP → обхода нет. Проверять по факту после установки.

**Риск (общий для всех иностранных eSIM в РФ):** при первом подключении к россети — блок интернета и SMS ~24 часа (гос. «cooling-off», описано в т.ч. самим BitJoy). Для Can некритично (телефон лежит дома, спешки нет).

### Рассмотренные операторы
- **BitJoy** (крипта, без KYC) — **отпал.** Сайт из РФ не открывается даже под VPN (1-я ссылка), 2-я только под VPN. Поиск «russia» дал тарифы с покрытием Russia (вероятно росIP = нет обхода), тарифы месячные пакетные ($6.49 unlimited / $24.99 20ГБ / $64.49 50ГБ). Независимых РФ-отзывов нет. Покупка требует VPN → ломает весь смысл «eSIM вместо VPN».
- **Unisim** — **текущий фаворит.** Крипта (BTC/ETH, заявлено; USDT уточнить), несгораемый баланс (помегабайтно ~$1.78/ГБ — долгоживущая), сайт/оплата доступны из РФ, 170+ стран, только интернет.

### Важный нюанс по обходу блокировок (честный вывод)
Основательница самого Unisim (Анна Кашницкая, интервью Hi-Tech Mail, апр 2026) публично говорит: иностранные симки — **ненадёжный** способ обхода в РФ. Зарубежный IP они дают, блокировки обходят, **но только пока россеть разрешает роуминговый канал**; канал могут резать целиком (отсюда нестабильность и 24ч-провалы). vc.ru: «не гарантированный обход».
→ Вывод: как эксперимент + международная симка на будущее — Unisim отличен. Как **надёжная замена VPN** — гарантий не даёт ни один оператор.

### Не доведено до конца (следующий шаг)
Сверить на живом сайте unisim.net: какие крипто-монеты (есть ли USDT), актуальная цена/ГБ, несгораемость баланса, и взять именно **глобальный** профиль (не «eSIM для России», иначе IP будет росс.). Пробовать через web_fetch, при JS-рендере — просить Can зайти.
