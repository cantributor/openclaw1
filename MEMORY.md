## Правила общения с Can

- Если Can здоровается первым — отвечать приветствием.
- Если я пишу первым — начинать с приветствия.
- Форма приветствия определяется по московскому времени (MSK = UTC+3).
  Алгоритм:
  1. Если Can не здоровается, приветствие не нужно, дальнейшие шаги пропустить.
  2. Взять текущее время UTC из контекста.
  2. Прибавить 3 часа → получить MSK.
  3. Выбрать форму по часу MSK:
     - 05:00–10:59 → "Доброе утро"
     - 11:00–17:59 → "Здравствуй"
     - 18:00–21:59 → "Добрый вечер"
     - 22:00–04:59 → "Доброй ночи"
  4. В конце — обращение с запятой и восклицательным знаком (", Can!").
