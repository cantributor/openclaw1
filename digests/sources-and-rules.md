# Ежедневный дайджест для Can — источники и общие правила

Cron `daily-crypto-ai-digest`, 07:00 МСК. Три отдельных поста подряд: Crypto, AI, State.

## Источники

Ленты через `web_fetch` от `https://t.me/s/<channel>`:

- https://t.me/s/forklog — крипта, общие новости
- https://t.me/s/forklogAI — ИИ
- https://t.me/s/bitsmedia — крипта
- https://t.me/s/vpnmagnum — РКН, блокировки, VPN, госрегулирование РФ
- https://t.me/s/vpn_perec — РКН, блокировки, VPN, госрегулирование РФ

## Общие правила

1. Отбирай посты за последние 24 часа.
2. **Разделение тем (без дублирования):**
   - **Crypto** — крипто-индустрия, технологии, экономика, рынок, госрегулирование крипты **в других странах** (не РФ).
   - **AI** — ИИ-индустрия, технологии, экономика, влияние на общество, прогнозы.
   - **State** — госрегулирование крипты **в РФ**, блокировки, РКН, цифровая слежка, законы РФ о VPN/интернете, громкие судебные дела в РФ, аресты, действия гос.органов РФ.
3. **Глубокий анализ и важные события в отрасли — высокий приоритет.** Развёрнутые аналитические материалы, мнения экспертов, прогнозы влияния на экономику/общество/политику, действия госорганов, крупных компаний, известных личностей. 
4. **Низкий приоритет / кратко или пропускать:** курсы криптовалют, общий sentiment рынка, рутинные новости о взломах.
5. **Из vpnmagnum и vpn_perec бери ТОЛЬКО собственные новости** — про уязвимости, цифровую слежку, госрегулирование, блокировки. НЕ бери: розыгрыши, лотереи, призы, приколы, шутки, шуточные новости.
6. **Дедуп между каналами** — один пункт, ссылка из forklog (для Crypto/AI) или из подходящего канала (для State).
7. **Ссылки — ОБЯЗАТЕЛЬНЫ в каждом пункте.** Иерархия:
   1. Оригинальный источник (forklog.com, bits.media и т.п.) — если упомянут в посте.
   2. Если оригинала нет — ссылка на сам тelegram-пост (`https://t.me/<channel>/<id>`); она доступна **всегда**, т.к. посты берутся из лент t.me/s/<channel>.
   
   Пункт без ссылки недопустим. Если по объёму ссылка не помещается — сокращай текст пункта, а не убирай ссылку.
8. **Формат каждого поста:** заголовок секции + подзаголовки по темам + пункты.
9. **Жёсткий лимит на каждый пост: 1500 символов.**
10. Отправляй три отдельных сообщения в Telegram-чат Can подряд: Crypto → AI → State. Не используй NO_REPLY.
