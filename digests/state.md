# Дайджест State

См. общие правила и источники: `/home/can/.openclaw/workspace/digests/sources-and-rules.md`.

## Фокус

Решения и действия **госорганов РФ** по поводу интернета и крипты: РКН, блокировки, цифровая слежка, законы о VPN/интернете, **госрегулирование крипты со стороны РФ** (ЦБ РФ, Минфин, ФНС, Госдума, СФ).

**Критерий:** новость о решении или действии именно российских госорганов. Решения иностранных компаний или зарубежных регуляторов, даже если они касаются россиян (выход биржи из РФ и т.п.), — это Crypto, не State.

## Источники для этой секции

- vpnmagnum
- vpn_perec
- forklog/bitsmedia (только посты про **решения российских госорганов** по крипте: ЦБ РФ, Минфин, ФНС, Госдума, СФ; российские законы о криптовалютах, российские налоги на крипту)

## Что НЕ включать

- Госрегулирование крипты в других странах → `crypto.md`.
- Решения иностранных компаний, касающиеся российских пользователей (выход из РФ, закрытие доступа и т.п.) → `crypto.md`. Сюда включаем ТОЛЬКО решения российских госорганов.
- Розыгрыши, лотереи, призы, приколы, шутки, шуточные новости из vpnmagnum/vpn_perec — игнорировать полностью.
- Реклама VPN-сервисов — игнорировать.
